The repository includes representative Barba-CV JSON payloads for different CV profiles.

- [`barba-cv.template.json`](https://github.com/Eurobotics-Association/barba-cv/blob/main/examples/barba-cv.template.json) — Minimal template showing the target structure and field organization.
- [`Clara_Delaunay_International_Internship_20260316.json`](https://github.com/Eurobotics-Association/barba-cv/blob/main/examples/Clara_Delaunay_International_Internship_20260316.json) — Internship-oriented profile with academic trajectory and multilingual details.
- [`Marc_Valentin_Customer_Service_Manager_20260316.json`](https://github.com/Eurobotics-Association/barba-cv/blob/main/examples/Marc_Valentin_Customer_Service_Manager_20260316.json) — Customer-service management profile with operational and leadership experience.
- [`Nassim_Karoubi_Business_Development_Internship_20260316.json`](https://github.com/Eurobotics-Association/barba-cv/blob/main/examples/Nassim_Karoubi_Business_Development_Internship_20260316.json) — Business-development internship profile with sales and growth-oriented positioning.

Use these examples as mapping references when generating or validating Barba-CV payloads.
